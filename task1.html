Midterm Lab Task 1 - Data Cleaning and Preparation using Excel

It means cleaning and preparing data in Excel by fixing errors, filling missing values, and organizing information for analysis.

View on GitHub

Step by Step

Step 1 Identify Dirty Data

Spot missing values like blank revenue or payment fields. Check formula errors and inconsistent names in the dataset.

Step 2 Map IDs to Real Labels

Match fields like client, department, state, and payment IDs with their correct labels from the reference sheet.

Step 3 Clean Values

Replace blanks with correct entries and fix invalid values so the data becomes consistent and easier to analyze.

Step 4 Prepare Final Dataset

Review the cleaned worksheet and confirm that the final dataset is complete and ready for reports or further analysis.
